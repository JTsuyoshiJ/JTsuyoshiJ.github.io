Minesweeper

Правила игры

Цель игры — открыть все клетки на поле, не попав на мину. При открытии клетки показывается число, обозначающее количество мин в соседних клетках. Игрок может помечать клетки с предполагаемыми минами флажками.

Инструкция:

  1. Выберите размер поля и сложность игры.
  2. Нажимайте на клетки, чтобы открыть их.
  3. Используйте правую кнопку мыши, чтобы помечать клетки флажками.
  4. Откройте все безопасные клетки, чтобы выиграть. Если вы откроете клетку с миной, игра будет проиграна.
